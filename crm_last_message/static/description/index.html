🧠 CRM Last Message on Leads

See the last message at a glance in list and form views

This module adds a computed field to CRM Leads that shows the last incoming message. It makes it easier for your sales team to see the latest customer interaction without opening the full record.

✨ Features

  • 📩 Displays the last incoming message on CRM leads
  • 📝 Clean HTML to plaintext preview with character limit
  • 👀 Viewable in both list and form views
  • 🧩 Simple installation and no configuration required

🚀 Compatible with Odoo 18.2

Tested and optimized for Odoo 18.2. No external dependencies.
